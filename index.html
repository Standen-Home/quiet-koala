Quick start (teacher)
  1. Click Start / Allow Microphone
  2. Click Calibrate QUIET (10s)
  3. Click Calibrate TOO LOUD (5s)
  4. Pick a character
  5. (Optional) click Full screen and Lock settings so students can’t change it
Tip: if you want this to work offline, click Install (Chrome/Edge) once it appears.
On iPad/iPhone: Share → Add to Home Screen.
Note: microphone access needs https:// or http://localhost.
Manage character packs (teacher)
Paste a characters/index.json-style payload below. This is stored on this device only (localStorage) and overrides the built-in pack list.
Tip: use this to hide characters, rename them, or reorder options.
Safety: only SVGs that are packaged with the app should be referenced.
